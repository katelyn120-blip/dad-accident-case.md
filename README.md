# Dad’s Accident Case

This repository is the master organization hub for my dad’s motor vehicle accident case.

## Purpose
This repo is used to:
- keep facts organized
- track documents and communications
- separate verified facts from analysis
- identify missing evidence and next steps
- reduce overwhelm

## Main Files
- `master-case-file.md` — the main verified case summary and timeline
- `document-index.md` — list of key documents, dates, and significance
- `communications-log.md` — record of calls, emails, and follow-ups
- `open-questions.md` — unresolved issues, missing evidence, and items needing confirmation

## Rules
- keep facts separate from assumptions
- do not treat analysis as verified fact
- verify before acting
- keep communications and next steps logged
